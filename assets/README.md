# Candour Offsite Presentation — Assets Folder

Place your images and videos in this folder. The website is pre-configured to detect them by name. If an image is present, it will automatically load and replace the glass placeholder on the website!

## Expected Filenames & Dimensions:

1. **Hero Banner / Video**:
   * File Name: `hero-banner.jpg` (or change to video inside `index.html`)
   * Recommended Size: `1200 x 900px` (16:9 ratio)

2. **The Shift Slider Cards**:
   * Card 1: `shift-1.jpg` (Recommended: `600 x 600px` or square JPG/PNG)
   * Card 2: `shift-2.jpg` (Recommended: `600 x 600px`)
   * Card 3: `shift-3.jpg` (Recommended: `600 x 600px`)

3. **Creative Memory**:
   * File Name: `memory-archive.jpg`
   * Recommended Size: `800 x 600px`

4. **Culture Studio**:
   * File Name: `culture-studio.jpg`
   * Recommended Size: `800 x 900px` (vertical ratio)

## Supported Formats:
* Images: `.jpg`, `.png`, `.webp`, `.svg`
* If you want to use a different format (like `.png` instead of `.jpg`), simply open `index.html` and edit the `src` attribute of the corresponding `<img>` tag.
